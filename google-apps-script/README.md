# 율량중학교 급식 설문 자동 동기화

`MealFormSync.gs`는 율량중학교의 오늘 급식을 공식 나이스 급식 API에서 조회하여, Google Forms의 객관식 또는 드롭다운 질문 선택지로 자동 반영합니다.

## 고정 학교 식별값

- 교육청 코드: `M10` (충청북도교육청)
- 학교 코드: `8011092` (율량중학교)
- 나이스 서비스: `mealServiceDietInfo`

## Apps Script에 추가

1. Google Forms 응답이 연결된 Google Sheets를 엽니다.
2. `확장 프로그램 → Apps Script`로 들어갑니다.
3. 왼쪽 `+ → 스크립트`를 눌러 파일 이름을 `MealFormSync`로 만듭니다.
4. `MealFormSync.gs` 전체를 붙여 넣고 저장합니다.
5. 기존 차트용 `Code.gs`와 `Index.html`은 그대로 둡니다.

## 최초 실행

Apps Script 상단 함수 목록에서 `setupMealFormSync`를 선택해 1회 실행합니다.

이 함수는 다음 작업을 수행합니다.

- 응답 스프레드시트에 연결된 Google Form 자동 탐색
- 제목에 `기대되는 급식 메뉴`가 들어간 객관식/드롭다운 질문 탐색
- 매일 오전 6시 전후 자동 실행 트리거 생성
- 오늘 급식 선택지 즉시 반영
- 매일 같은 Form에 다시 응답할 수 있도록 `응답 1회 제한` 해제

최초 실행 시 Google Sheets, Google Forms, 외부 API 호출, 트리거 생성 권한 승인이 필요합니다.

## 질문 제목

다음처럼 `기대되는 급식 메뉴`가 포함되어야 합니다.

```text
오늘 가장 기대되는 급식 메뉴는?
```

질문 유형은 다음 중 하나여야 합니다.

- 객관식
- 드롭다운

## 테스트 함수

### 오늘 데이터만 확인

```text
previewTodayMealChoices
```

Form은 수정하지 않고 실행 로그에 오늘 메뉴를 출력합니다.

### 오늘 메뉴를 실제 Form에 반영

```text
syncTodayMealToForm
```

### 주말·방학 테스트

```text
syncNextAvailableMealToForm
```

오늘부터 14일 안에서 가장 가까운 급식일을 찾아 실제 Form에 반영합니다. 이 함수는 테스트 목적으로 미래 날짜 선택지를 미리 열 수 있으므로, 테스트 후 평일에는 `syncTodayMealToForm`을 다시 실행합니다.

## 자동 동작

매일 오전 6시 전후 `scheduledMealFormSync`가 실행됩니다.

- 급식이 있으면 메뉴 선택지를 교체하고 응답을 엽니다.
- 급식이 없으면 해당 날짜 안내문을 표시하고 응답을 닫습니다.
- API 오류가 발생하면 기존 선택지를 덮어쓰지 않고 실행 오류로 남깁니다.

Apps Script 시간 기반 트리거는 지정 시각에 정확히 실행되는 것이 아니라 해당 시각 전후로 실행될 수 있습니다.

## 일일 응답 제한 주의

Google Forms의 `응답 1회로 제한`은 하루 1회가 아니라 **그 Form 전체에서 계정당 1회**입니다. 같은 설문지를 매일 재사용하려면 이 설정을 끄는 것이 필요합니다. `setupMealFormSync`가 자동으로 해제합니다.

현재 차트는 오늘 날짜의 응답만 집계하므로 이전 날짜의 시트 데이터는 차트에 섞이지 않습니다. 중복 투표를 더 엄격히 막으려면 이메일 수집 후 같은 날짜·같은 이메일의 마지막 응답만 집계하도록 차트 코드를 추가 개선할 수 있습니다.

## 선택적 나이스 API 키

기본 호출은 키 없이도 동작하도록 되어 있습니다. 별도 나이스 API 키를 사용할 경우 Apps Script의 `프로젝트 설정 → 스크립트 속성`에 다음을 추가합니다.

```text
속성: NEIS_API_KEY
값: 발급받은 API 키
```
