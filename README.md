# 율량중학교 오늘의 급식

율량중학교의 급식 정보를 나이스 교육정보 개방 API에서 불러와 보여주는 반응형 웹 위젯입니다.

## 주요 기능

- 오늘 및 선택 날짜 급식 조회
- 이전 날·다음 날·오늘 이동
- 주간 날짜 선택과 급식 유무 표시
- 메뉴별 알레르기 번호 및 칼로리 표시
- 보라색 파스텔톤 반응형 디자인
- Google Sites URL 임베드 지원
- API 키를 브라우저에 노출하지 않는 Vercel 서버리스 함수

## 배포

1. Vercel에서 **Add New → Project**를 선택합니다.
2. `sh85952223/bobbobbob` 저장소를 Import합니다.
3. 별도 빌드 설정 없이 **Deploy**를 누릅니다.

선택 사항으로 나이스 인증키가 있다면 Vercel 환경 변수에 `NEIS_API_KEY`를 등록할 수 있습니다.

## 구조

- `index.html`: 화면, 디자인, 날짜 이동 로직
- `api/meal.js`: 나이스 학교 검색 및 급식 조회 API
- `vercel.json`: 서버리스 함수 캐시 및 보안 헤더

급식 조회 흐름은 `school-kr`의 학교 검색 → 월별 급식 조회 구조를 참고했으며, 데이터 소스는 현재 공식 나이스 교육정보 개방 API를 사용합니다.
